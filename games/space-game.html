Controls:

Astronaut: Controller --Left Stick to move, X to increase tether, B to decrease tether; Face buttons (A-B-X-Y) for quick-time input. Keyboard: I, J, K, L to move; these mappings happened very last minute due to build errors, so tether controls and quick-time input were not able to be sufficiently mapped in time.

Spaceship: A on keyboard to turn counterclockwise, D on keyboard to turn clockwise; space-bar to move forward

*All art assets were of our own creation. We adapted online C# scripts for a physics-based spring tether and cell-based procedural generation algorithm in order to save time. All other code is of our own creation. We decided not to continue to work on this game after the jam so that we could use it as an example of an ambitious weekend project.
