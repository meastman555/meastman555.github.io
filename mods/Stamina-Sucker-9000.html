Stamina Sucker 9000

Because who doesn’t want a giant — yet fast — war-hammer to drain an opponent’s energy?

The first mod I made for Skyrim. This weapon was custom created using in-game assets. All I did was fiddle with a standard Orchish war-hammer’s values and properties, created a custom, powerful enchantment by duplicating an existing one and changing its description and properties, attached it to my new weapon, then loaded it into an in-game cell. Quite simple, but I learned the basics of the Creation Kit and how assets/weapons/enchantments and their related properties worked within the engine.
